# Mitwirken (Contributing)

Vielen Dank für Ihr Interesse an der Roboter-Umgebung!

## Entwicklungsprozess

1. Forken Sie das Repository.
2. Erstellen Sie einen neuen Feature-Branch.
3. Implementieren Sie Ihre Änderungen.
4. Fügen Sie Tests für neue Funktionen hinzu.
5. Stellen Sie sicher, dass alle Tests bestehen.
6. Erstellen Sie einen Pull Request.

## Code-Stil
Wir verwenden:
- **Black** für die Code-Formatierung.
- **Ruff** für das Linting.
- **Google-Style Docstrings** für die Dokumentation.

## Tests ausführen
```bash
python3 -m pytest
```
